

# High-Impact Quick Tweaks

Here are practical, easy-to-implement improvements that will make the dashboard significantly more useful day-to-day:

---

## 1. Confirmation + Undo on Delete Actions
**What:** Every delete button across all modules (Tasks, Events, Expenses, etc.) currently deletes instantly with no safety net. Add a confirmation toast with an "Undo" button that gives you 5 seconds to reverse accidental deletes.
**Impact:** Prevents data loss — huge quality-of-life improvement.

## 2. CSV Import for Income (not just Expenses)
**What:** You already have CSV import for expenses. Mirror the same feature for the Income tab on the Budget page.
**Impact:** Saves manual data entry time, especially for bulk historical data.

## 3. Duplicate Item Action
**What:** Add a "Duplicate" button on Tasks, Events, and Expenses. One click creates a copy with "(Copy)" appended to the title.
**Impact:** Speeds up creating similar recurring items — very common workflow.

## 4. Quick Inline Status Change
**What:** On the Dashboard Home's task list and the Task Manager table, make the status badge clickable to cycle through statuses (Not Started → In Progress → Complete) without opening the full edit dialog.
**Impact:** Dramatically faster daily task updates.

## 5. Data Export Across All Modules
**What:** Tasks already have CSV export. Add the same export button to Events, Classes, Instructors, and Partnerships pages.
**Impact:** Makes reporting and data backup easy across the board.

## 6. "Welcome Back" Summary
**What:** On Dashboard Home, add a small top banner: "You have X overdue tasks, Y due this week, Z new items since last visit" with direct links.
**Impact:** Instant situational awareness when you open the app.

---

## Estimated Effort
Each tweak is 15-30 minutes of implementation. I'd suggest picking 3-4 to start with. Which ones do you want me to build?
